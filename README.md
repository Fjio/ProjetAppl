# PGROU

## Outils utilisés

- Symfony CLI v4.13.3
- Composer v1.10.1
- PostgreSQL v11.5

## Installation des dépendances

A la racine du projet :

```bash
composer install
```

## Base de données
pgrou

## Lancement du serveur

Pour lancer le serveur, exécuter à la racine du projet :

```bash
symfony server:start
```
